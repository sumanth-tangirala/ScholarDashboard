# Glossary

Research terms, methods, and shorthand relevant to Sumanth's work.

## Methods & Techniques
| Term | Meaning | Context |
|------|---------|---------|
| Conformal prediction | Distribution-free uncertainty quantification framework | Key method interest |
| Active learning | Querying strategies to efficiently label/collect data | Key method interest |
| Diffusion models | Score-based generative models via iterative denoising | Used in verification, IL, generalist policies |
| Flow matching | Generative modeling via continuous normalizing flows | Used in verification, IL, generalist policies |
| SBMP | Sampling-based motion planning | Past research (RRT, PRM, etc.) |
| RRT | Rapidly-exploring Random Trees | Motion planning algorithm |
| PRM | Probabilistic Roadmap | Motion planning algorithm |
| IL | Imitation learning | Learning from demonstrations |
| RL | Reinforcement learning | Learning from reward signals |
| HJ | Hamilton-Jacobi (reachability) | Related to verification/safety |
| CBF | Control Barrier Function | Safe control method |
| CLF | Control Lyapunov Function | Stability/control method |

## Application Areas
| Term | Meaning | Context |
|------|---------|---------|
| Black-box verification | Verifying system properties without model access | Core research |
| Controller composition | Combining multiple controllers with guarantees | Research interest |
| Generalist policies | Multi-task robot policies (e.g., RT-X, Octo, pi0) | Active interest |
| Safe control | Control with formal or statistical safety guarantees | Research interest |
| Robust control | Control that handles uncertainty/perturbations | Research interest |
